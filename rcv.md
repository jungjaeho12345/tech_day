# rcv.md
수집(자동기사)기 기술 명세서이다.
데이터를 API 사용 또는 FTP로 수신받아 해당 파일을 파싱하여 기사로 등록한다.

### 자동기사 명세서
수신받은 데이터에서 제목, 본문내용을 자동으로 추출하여 데이터베이스 테이블 Content, Article에 입력한다.

### 규칙
 - 자동기사는 데이터베이스 속성에 '자동기사'를 꼭 입력한다.
 
## 수신 명세서
 - 등록되어 있지 않은 ID는 데이터를 수신받지 않는다.
 - FTP로 수신받을 수 있도록 구현한다. event 방식이며 수신받은 파일을 분석한다.
 - API 호출 후 응답받은 데이터를 분석한다.

## 분석 명세서
 - FTP에서 수신받은 파일을 제목, 본문내용을 분석한다.
 - API 호출 후 전달받은 데이터를 제목, 본문내용을 분석한다.

## 등록 명세서
 - 분석데이터를 DB Article, Contents 입력한다.

## 관리 명세서
 - API 설정, FTP 송신, 수신을 설정할 수 있는 페이지(rcvMgmt.do)를 조회/생성/삭제 할 수 있다.
