# Project Interview

## Round 1: Vision
Question: 이 프로젝트는 무엇을 하며 누구를 위한 것입니까?
Answer: 언론사 기사 제작 시스템. 기자·데스크·관리자가 기사를 작성·수정·송고·조회하는 사내 기사 CMS의 '제작' 시스템이다. news.md 명세 기반.

## Round 2: Technology
Question: 주요 기술 스택은 무엇입니까?
Answer: 서버는 NodeJS + MVC 패턴 + SQLite DB. 클라이언트는 React + Vite + MVC 패턴.

## Round 3: Scope
Question: 핵심 기능과 경계는 무엇입니까?
Answer: 제작 시스템만 진행한다. 기사/사용자 CRUD 함수, 로그인, 기사 생애주기(RDS/DPS/RRH 등), 기사 작성·조회 페이지 포함. 수집·배부 시스템은 현재 범위에서 제외(향후 확장).
